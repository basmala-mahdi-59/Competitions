# My PHP Project

## Description
This is a web project built using PHP, HTML, CSS, JavaScript, and a database.  
It demonstrates a full-stack web application with dynamic content and user interaction.

---

## Technologies Used
- Frontend: HTML, CSS, JavaScript
- Backend: PHP
- Database: MySQL , XAMPP

---

## Installation & Setup
Follow these steps to run the project locally:

1. Download or clone the repository:
   `bash
   git clone https://github.com/basmala-mahdi-59/Competition.git
2. Copy the project files to your XAMPP htdocs folder.
3. Start Apache and MYSQL in XAMPP.
4. At project folder paste at URL:
   localhost/Competitions
5. Import the provided SQL file into phpMyAdmin.
